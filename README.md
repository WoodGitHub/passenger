# passenger
Passenger Mod for Garry's Mod 13

Features:
- Horn
- Seat Ejection
- Anti Stuck System
- Nearest seat finder
- Lock/ Unlock seats
- Hydraulic Support

Console Commands:
Eject - Eject all passengers
Lock - Locks all seats
Unlock - Unlocks all seats

-- Created by Wood, fixed for Garry's Mod 13 then released. --
